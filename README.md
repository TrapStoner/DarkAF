# DarkAF
> [!IMPORTANT]
> Change Discord's theme to *Dark* in Appearance settings if you are using *Light mode*!

 **[Download](DarkAF.theme.css)**
 - Place it in `%appdata%/BetterDiscord/Themes`
 - Go to BetterDiscord Settings > Themes and enable *DarkAF*
